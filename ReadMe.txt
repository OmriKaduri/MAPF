-------   MAPF   -------

- To choose which MAPF instances to solve go to Program.cs, Main() method. 
- You can choose to run either grids, dragon age maps, mazes, or a specific instance.
- Choose also the number of instances, number of agents, grids size, and obstacles percents.

- To choose solvers (algorithms), go to Run.cs, Run() method. Uncomment which solver to add to the solvers list.

- Note that the program will create instances in the instances folder.
